# 생활속안전 안드로이드 앱


URL : https://play.google.com/store/apps/details?id=info.project.datapotal

사용된 라이브러리 목록
* 이미지 캐싱처리 : AUIL
* 리스트뷰 애니메이션 효과 : Jazzy
* 메뉴효과 : menuDrawer
* 도움말 ViewPager 부분 : viewflow

사용된 기술 목록
* ViewPager 
* BaseAdapter
* JSON Parsing
* SQL_Lite
* Intent
* Splash View
* infinite scrolling
* FrameLayout is placed
* inflate
* Holder
